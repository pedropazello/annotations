# Ruby annotations
## How to create .ruby-version and .ruby-gemset
`rvm --create --ruby-version ruby-1.9.3@my-gemset`

## .editorconfig for Ruby apps
```
root = true

[*]
indent_style = space
indent_size = 2
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true

[*.{sh,markdown}]
indent_size = 4

```
